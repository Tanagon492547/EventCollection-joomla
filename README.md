"# EventCarousel-joomla" 

###### สำหรับ mod_event_collection เป็น Module ของ joomla โดยพัฒนาจาก joomla 5 โดยเฉพาะ

## วิธีนำไปใช้
*ให้ เลือกไฟล์ mod_event_collection.zip ไปใส่ในเว็บไซต์  joomla ของคุณ*

## Filed ที่ใช้
*เรามาทำความรู้จัก filed ที่ใช้ ใน module นี้กัน ว่ามีอะไรบ้าง*

| Filed  | คำอธิบาย | ประเภท |
| ------------- |:-------------:|:-------------:|
| url-event      | ลิงค์ | url |
| image-event     | ภาพ | media |
| description-event | คำอธิบาย | text |

## เลือก Group Filed 

*ตอนที่สร้าง filed ให้เลือก group ให้มันด้วย*

## กำหนด Articles

*ให้ไปที่ Articles ที่ต้องการเเล้วดูส่วนบน หาชื่อ Group Filed ที่คุณสร้างไว้เเล้วก็เพิ่มข้อมูลเข้าไป*

## นำไปใช้กับ site module
*ให้สร้าง Categories เเล้วไปคลิก site module Event Collection PSU เเล้วเลือก Categories ที่คุณสร้าง*
